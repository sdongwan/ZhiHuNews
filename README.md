实现功能：
	1、新闻详情页折叠式标题栏的实现（采用android 5.x 兼容性控件）<br/>
	2、recycleview的上拉刷新，下拉加载实现 <br/>
	3、recycleview的不同item类型布局的实现  <br/>
	4、webview加载网页的时候，采用android与js交互，实现点击图片，跳到查看图片的acitivity，实现图片的放大和缩小查看
